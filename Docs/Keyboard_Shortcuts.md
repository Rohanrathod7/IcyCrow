# Keyboard Shortcuts Reference

IcyCrow provides global keyboard shortcuts for efficient research and annotation.

| Shortcut | Action |
| :--- | :--- |
| **V** | Select Tool |
| **H** | Pan / Hand Tool |
| **M** | Highlighter |
| **P** | Pen / Draw |
| **E** | Eraser |
| **T** | Text Tool |
| **S** | Sticky Note |
| **C** | Callout |
| **Ctrl/Cmd + S** | Save Workspace |

> [!NOTE]
> Shortcuts are automatically disabled when typing in text fields (Inputs, Textareas, or Callout boxes) to prevent accidental tool switching.
